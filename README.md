# Week 3 Day 1 - Machine Learning Fundamentals

## Objective

The objective of this project is to understand the complete Machine Learning workflow and build a predictive model using Linear Regression in Scikit-learn.

## Dataset

Student Performance Dataset containing:

* Study_Hours
* Attendance
* Assignments
* Marks

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Plotly
* Jupyter Notebook
* VS Code

## Tasks Completed

* Dataset Exploration
* Features and Label Identification
* Machine Learning Workflow Documentation
* Train-Test Split
* Linear Regression Model Training
* Predictions Using New Data
* Actual vs Predicted Comparison
* Model Evaluation
* Data Visualization using Plotly
* Student Research Activity

## Evaluation Metrics

* Mean Absolute Error (MAE)
* R² Score
