1. Data Collection

Data collection is the first step in the machine learning workflow. In this step, relevant data is gathered from different sources. For this assignment, a student performance dataset was provided. The dataset contains study hours, attendance, assignments, and marks. Good quality data helps build an accurate model.

2. Data Cleaning

Data cleaning ensures that the dataset is free from errors and inconsistencies. Missing values, duplicate records, and incorrect entries are identified and corrected. Clean data improves model performance and reliability. In this dataset, no missing values were present.

3. Feature Selection

Feature selection involves choosing the input variables that influence the target variable. In this project, Study_Hours, Attendance, and Assignments were selected as features. Marks was selected as the target variable. Proper feature selection improves prediction accuracy.

4. Train-Test Split

The dataset is divided into training and testing sets. The training set is used to teach the model, while the testing set evaluates its performance. In this project, 80% of the data was used for training and 20% for testing. This helps measure how well the model performs on unseen data.

5. Model Training

Model training is the process of learning patterns from the training data. A Linear Regression algorithm was used in this assignment. The model analyzed the relationship between the selected features and marks. After training, it was able to make predictions.

6. Testing

The trained model was tested using the testing dataset. Predictions were generated for the unseen records. The predicted values were then compared with the actual values. This helps determine the effectiveness of the model.

7. Evaluation

Model evaluation measures how accurately the model performs. Metrics such as Mean Absolute Error (MAE) and R² Score were calculated. MAE measures prediction error, while R² Score measures how well the model explains the data. Better metrics indicate a stronger model.

8. Prediction

After evaluation, the model was used to predict marks for new student records. The prediction process uses learned patterns from the training data. These predictions can help estimate future outcomes. This is the final goal of machine learning.
